# md

导出内存区域内容。

## 用法

```
md [-b|-w|-l|-q] address [-c=count]
```

## 选项

| 选项 | 说明 |
|------|------|
| `-b` | 字节（8位，默认） |
| `-w` | 字（16位） |
| `-l` | 长字（32位） |
| `-q` | 四字（64位） |
| `-c=count` | 显示元素数量（默认 64） |

## 说明

- 显示十六进制值和 ASCII 表示
- 遵守访问大小的对齐约束
- 支持 Ctrl-C 中止

## 示例

```bash
# 以字节显示 64 字节
md -b 0x80000000

# 以长字显示 16 个值
md -l 0x80000000 -c=16
```
