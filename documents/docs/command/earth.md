# earth

显示一个旋转的 3D 贴图地球。使用查表法 (`fasin`/`fatan2`) 进行球面渲染，支持鼠标和触摸交互。未指定贴图时使用内置的 1024×512 JPEG 地球纹理。

## 用法

```
earth [texture] [-f=framebuffer] [-i=input]
```

## 选项

| 选项 | 说明 |
|------|------|
| `texture` | 可选的地球贴图文件（等距矩形投影），省略时使用内置纹理 |
| `-f=<fb>` | 目标帧缓冲设备 |
| `-i=<input>` | 输入设备 |

## 说明

- 贴图通过 XFS 加载；文件解析成功但加载失败时会报错退出，不再静默回退到内置图像
- 默认自动水平自转，约每 120 秒一周
- 支持鼠标拖拽旋转、滚轮缩放
- 支持单指触摸拖拽旋转、双指捏合缩放
- 缩放范围限制在 0.2 ~ 5.0 倍
- 纬度旋转限制在 ±90° 之间
- 约 60 FPS 刷新，按 Ctrl-C 退出

## 示例

```bash
# 使用内置纹理
earth

# 指定帧缓冲与输入设备
earth -f=fb.0 -i=input.0

# 使用自定义贴图
earth earth_texture.jpg
```
