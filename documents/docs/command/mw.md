# mw

写入值到内存区域。

## 用法

```
mw [-b|-w|-l|-q] address value [-c=count]
```

## 选项

| 选项 | 说明 |
|------|------|
| `-b` | 字节（8位，默认） |
| `-w` | 字（16位） |
| `-l` | 长字（32位） |
| `-q` | 四字（64位） |
| `-c=count` | 写入次数（默认 1） |

## 说明

- 将同一值重复写入连续物理内存
- 遵守访问大小的对齐约束
- 支持 Ctrl-C 中止

## 示例

```bash
# 写入一个字节
mw -b 0x80000000 0xff

# 写入 16 个长字
mw -l 0x80000000 0xdeadbeef -c=16
```
